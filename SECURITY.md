# Security Policy

## Reporting a Vulnerability

Please report security issues privately to:

- Santiago Garcia
- `santi.gamo@gmail.com`

Include:

- Affected endpoint(s) or component(s)
- Reproduction steps or proof of concept
- Impact and severity assessment (if known)
- Any suggested mitigations

Do not open public GitHub issues for potential vulnerabilities.

## Supported Versions

Formal version support windows are not defined yet. Report against the latest `main` branch.
